Methods available

Id Method Description
eof Ezeoleaf Characters and numbers
vne Vigenere Characters and numbers
num Numeric Only number
sxf SixtyFour Characters and numbers with a minimum length of 64

Note

Cryptor comes with a default dictionary, but it can be changed following some steps:

  • Open GenerateDictionary.html
  • Click on Generate
  • Replace the given dictionary in the dictionary of the library (scripts/cryptor.js and scripts/cryptor.min.js)
